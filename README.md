# SmileBack - Course Feedback System

Anonymes Emoji-basiertes Live-Feedback System für Vorlesungen.

## Projekt starten

1. Terminal im Projektordner öffnen

2. Server starten:
```bash
python3 -m http.server 8000
```

3. Browser öffnen:
```
http://localhost:8000/
```

## Navigation

- **Student-Ansicht:** `student.html` - Feedback geben
- **Dozenten-Dashboard:** `lecturer.html` - Live-Feedback ansehen
- **Analytics:** `analytics.html` - Historische Daten

## Hinweis

Das System verwendet localStorage für die Datenspeicherung (Prototyp).
